Smartphone Shop


Project Description:

A modern e-commerce frontend for browsing and displaying smartphones with a clean UI and product-focused layout.


Tech Stack / Built With:

- React
- TypeScript
- Vite
- CSS Modules

Features:

- Product listing interface
- Responsive design
- Component-based architecture
- Clean UI layout


Getting Started / Installation:

- git clone https://github.com/danemajstorcev/smartphone-shop.git
- cd smartphone-shop
- npm install
- npm run dev

Live Demo:
https://smartphone-shop-phi.vercel.app/
